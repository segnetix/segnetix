Splitwatch v2.0 is a timing application that gives the serious track/swimming fan or coach a powerful interval stopwatch on their iPhone, or iPod touch. Splitwatch features a completely new interface updated for iOS 8 and iPhone 6 and 6 Plus.

  • Designed especially for Track and Swimming training and competition but is also effective for any event where interval recording is needed
  • Automatically calculates the split breakdown as you are timing, for example, if set for 200m intervals would yield each 200m, 400m, 800m, and 1600m (or 1000m if in Kilo split mode)
  • Record either metric or English interval distances: 25, 50, 100, 200, or 400 meters or 25, 50, 110, 220, or 440 yards
  • Time any number of competitors simultaneously with the Multiwatch mode, starting the watches at the same time or independently
  • Quickly recall any recorded timing event in the History view, including athlete name, event name, race start date and time, final time, splits, and split breakdown
  • Sort timing events by date, event distance, event name, or athlete name
  • Email a Split Report with all the info and split breakdown right from Splitwatch
  • Flashes each time the start/lap/stop buttons are hit for positive click feedback
  • Optionally produces a click sound for each start/lap/stop tap
  • Optionally disables the lap button for 2 seconds to prevent unintentional double-clicks
  • Split Editing Individual splits are fully editable, including add, change and delete. Also, you can change the base units and split modes for any event in your database. It is easy to fix timing glitches.
  • Lap Mode This mode allows you to record events where the intervals are not tied to a specific distance.
  • Furlong Mode Furlong mode is for horse racing. Distances are displayed in fractions of a mile instead of yards.
  • Running Time/Lap Time Swap Tap the Splitwatch main time display to swap the running time with the lap time.
  • More Event Summary Info In addition to closing splits, Splitwatch provides split summary info with Min, Max, and Avg lap times.

  • NEW FEATURES in v2.0:
  • New Interface Completely redesigned for iOS 8 and fully compatible with iPhone 4s, iPhone 5, iPhone 5s, iPhone 6, iPhone 6 Plus and iPod touch (5th gen)!
  • Unlimited Multiwatches Add as many watches as you want in Multiwatch mode!
  • Clear Timing Data New function to clear all the data from the timing database.
